# Pentacore Website

Pentacore is a modern fintech/payment-gateway website built with **Next.js 16**, **TypeScript**, **Tailwind CSS**, **shadcn-style components**, **@remixicon/react**, and **Nodemailer**.

This website is designed for a payment gateway business that provides Payin, Payout, UPI collection, settlements, API integration, webhook callbacks, and merchant payment services.

---

## Project Overview

The project includes a professional responsive UI for Pentacore with modern fintech styling, reusable components, smooth spacing, rounded cards, and a clean layout.

It is suitable for presenting Pentacore services, merchant benefits, pricing highlights, FAQs, contact information, privacy policy, and terms of service.

---

## Tech Stack

- Next.js 16
- React

- @remixicon/react

---

## Clone Project

```bash
git clone git@github.com:protize/pentacore.git
```

Go inside the project folder:

```bash
cd pentacore
```

---

## Install Dependencies

```bash
npm install
```

If `npm` is not installed, install it first:

```bash
npm install -g npm
```

---

## Run Locally

```bash
npm dev
```

Open in browser:

```txt
http://localhost:3000
```

---

## Build Project

```bash
npm build
```

---

## Start Production Server

```bash
npm start
```
